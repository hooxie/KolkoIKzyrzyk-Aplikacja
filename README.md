This was my very first academic project, created during my studies.
It is a simple, desktop-based Tic-Tac-Toe game developed in Python using the tkinter library for the graphical user interface.

The application includes basic game logic that handles player turns, detects winning combinations on a 3x3 grid, and highlights the winning row or a tie.
Additionally, it features a functional restart button that allows players to reset the board and start a new game instantly.
